Basic Guide


First learn the shortcut by pressing the H key on your keyboard as image below:

Digimon Information Window

1- Change Digimon name; can be changed if you have a NameSwap item.
2- Open the tittle window. Guide about Tittles click here.
3- Add STR (ATK + HP)
4- Add DEX (Battle Level + ATK)
5- Add CON (HP, VP, Defense)
6- Add INT (VP, Battle Level, Defense)
7- Points to add Basic stats remaining.
8- Points to add Skill points remaining.
9- Attribute Digimon
10- Traits window.
11- Rookie window.
12- Champion window.
13- Ultimate window.
14- Mega window.

Commands

To see it you just type ".cmds" in General Chat then it will appear as shown below:


.online: Show the total number of Players that are Online.

.eventmap: Teleport to Event Map, this place is where you can exchange Digi-Code V/E/L/H. Guide about Digi-Code click here.

.exit: Teleport to Digital Heaven.
.unstuck: Teleport to one point on each map.
.enableca: To bring up the card slash animation during battle. (For VIP players)
.disableca: To turn off the card slash animation during battle. (For VIP players)
.dp: Show Cash Shop commands.
.points: Show the amount of DP (Donate Points)
.ml: Change and choose a macro language
.time: To see how long u have been Online.
.showbm: Show BM Digimon have in ur starter.
.archivechange: Save the bm in ur digimon. (Need Blessing Buff, 1 Evolutor, 50m Bits Required)
.rbevo: Evolve to mega in Royal Base. (For Blessing Player)
.showpd: Only see your and pt drops. (For Blessing Player)
.showad: See all drops. (For Blessing Player)
.party [name]: Remote invite party. (For VIP players)
.trade [name]: Remote trade. (For VIP players)

Capture Digimon

U can use 'C' in In-Training digimons and using Force Control card to leave 1hp the digimon. That increase the % of capture, but its only waste time. Its more fast go try get a Blue Capture Net.

This Capture Net is divided into 3 types:

Blue Capture Net


Use to capture all Digimon in the In-Training stage. Can be obtained from mobs Rank Elite(E), Legend (L), Hero (H), Legendary Hero (LH) and World Boss (WB, the best drop from he).
For Location guide about In-Training click here.

Orange Capture Net


Use to capture Digimon in Rookie form. However only a few can be captured with this net. Digimon you can catch with this are Tapirmon, Dokunemon, Ryudamon, Dracmon and Kamemon. Only can be obtained from Cash Shop, or events.
In cash shop u can found Orange Capture Net (S), the diference its the digi u catch with this have a boost exp until lv 71.

Special Capture Net

Use to capture Digimon
